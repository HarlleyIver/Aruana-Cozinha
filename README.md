# Aruana Cozinha

Landing page da **Aruana Cozinha**, catering autoral e hospitalidade brasileira através da cozinha, em São Paulo.

Site de página única, feito para apresentar a marca, captar leads e servir de porta de entrada para os anúncios. Totalmente responsivo para computador e celular.

## O que tem aqui

- Identidade visual construída a partir da marca: fundo creme, terracota, os quatro elementos da natureza (ar, água, fogo, terra) e ilustrações botânicas em SVG no estilo dos grafismos da Aruana.
- Seções: hero, a história do nome Aruana, o que fazemos, os quatro elementos, tipos de encontro, como funciona e contato.
- Botão flutuante de WhatsApp em todas as telas.
- Formulário de contato que monta a mensagem e abre o WhatsApp já com o pedido pronto.
- Tipografia via Google Fonts (Fredoka, Caveat e Nunito).

## Como usar

O site é um único arquivo estático. Basta abrir o `index.html` no navegador.

Para colocar no ar, publique a pasta em qualquer hospedagem de sites estáticos, como Vercel, Netlify ou GitHub Pages. O `index.html` na raiz já é o ponto de entrada.
